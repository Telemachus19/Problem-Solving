# LeetCode 300-Problem Roadmap

This is the main day-by-day roadmap for the LeetCode folder. The target is 300 easy-to-medium problems with broad DSA coverage, and the solved-history section below stays as a separate reference for items already completed in the repository.

## Weekly Milestones

- Week 1: Days 01-07. Build the base with arrays, strings, hashing, sliding window, linked lists, stacks, and binary search.
- Week 2: Days 08-14. Move through intervals, greedy, trees, BST, heaps, backtracking, and graph traversal.
- Week 3: Days 15-21. Cover union-find, bit manipulation, and the first deep pass through dynamic programming and tree construction.
- Week 4: Days 22-30. Finish with matrix work, parsing, prefix sums, review patterns, shortest paths, design, and mixed review.
