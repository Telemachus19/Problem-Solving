# LeetCode 300-Problem Roadmap

This is the main day-by-day roadmap for the LeetCode folder. The target is 300 easy-to-medium problems with broad DSA coverage, and the solved-history section below stays as a separate reference for items already completed in the repository.

## 600 Target Plan (Apr 18 -> Jun 30)

Start date: 18 Apr 2026
End date: 30 Jun 2026
Window: 74 days

Target: 600 solved by 30 Jun using a two-lane day plan:

- Ladder lane (ascending topics): 2-5 new problems/day.
- Reinforcement lane (speed/review): 3-4 problems/day from previously studied patterns.
- Daily total solved target: 8 problems/day, with 9 problems on 8 push days.

Math:

- Base: 74 x 8 = 592
- Push days (+1 each): 8 days x 1 = 8
- Final target by 30 Jun: 600 solved in this window

### Ascending Topic Ladder

- 18 Apr -> 27 Apr: Arrays, strings, hashing, two pointers, sliding window, prefix sums, binary search basics.
- 28 Apr -> 10 May: Linked lists, stack/queue, monotonic stack, intervals, greedy, heap basics.
- 11 May -> 24 May: Trees/BST traversal, recursion, divide and conquer, backtracking fundamentals.
- 25 May -> 08 Jun: Graphs (BFS/DFS), topological sort, shortest paths, union-find, grid graphs.
- 09 Jun -> 20 Jun: Dynamic programming ladder (1D -> 2D -> subsequence/knapsack/state DP).
- 21 Jun -> 30 Jun: Mixed review, weak-topic repair, design/implementation, timed mocks.

### Weekly Execution Template

Weekly template for 18 Apr to 30 Jun:

- Default day: 4 ladder + 4 reinforcement
- Push day: 4 ladder + 5 reinforcement
- Low-energy day fallback: 2-3 ladder + remaining reinforcement

This keeps learning in ascending order while still building enough solve volume for the 600 target.
